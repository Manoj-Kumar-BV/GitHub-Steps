Creating a New Branch

Follow these steps to create a new branch on GitHub:

  1. **Create a New Repository**:
    • When you create a new repository on GitHub, it will start with the `main` or `master` branch (depending on your settings).
  2. **Clone the Repository to Your Local Machine**: 
    git clone https://github.com/YourUsername/YourRepository.git
cd YourRepository
  3. **Create a New Feature Branch**: 
    git checkout -b feature-name
  4. **Make Changes and Commit**: 
    git add .
git commit -m "Description of changes"
  5. **Switch to Main/Master Branch**: 
    git checkout main  # or 'master' if your branch is named 'master'
  6. **Pull Latest Changes from Remote Main/Master**: 
    git pull origin main  # or 'master' if your branch is named 'master'
  7. **Merge Feature Branch into Main/Master**: 
    git merge feature-name
  8. **Push Updated Main/Master Branch to Remote**: 
    git push origin main  # or 'master' if your branch is named 'master'
  9. **Delete the Feature Branch (optional)**: 
    git branch -d feature-name

Next Steps

After creating your branch, you can:

  • Create repositories to manage different projects.
  • Start committing code to your branches.
  • Collaborate with others by inviting them to your repository.
